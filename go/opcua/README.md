# OPC UA Go Experiments

Using the [Native Go OPC-UA library](https://github.com/gopcua/opcua)

## Testing

installing with either

```go get github.com/gopcua/opcua```

or

```go install github.com/gopcua/opcua@latest```

execute

```go run main.go```

## Building

execute

```docker build -t <name> .```
